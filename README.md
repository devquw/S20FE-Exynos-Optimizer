# S20 FE Exynos 990 Optimizer 🚀

A lightweight, systemless module designed to tame the **Exynos 990** on the Samsung Galaxy S20 FE. This module focuses on reducing heat and improving battery life by capping the power-hungry Mongoose M5 cores.

## ✨ Features
- **Thermal Control:** Caps Big cores (M5) at **1.8GHz** to prevent overheating.
- **Battery Friendly:** Reduces power consumption during gaming and daily tasks.
- **Governor Tweak:** Sets CPU governor to `schedutil` for a better balance between performance and efficiency.
- **Universal Compatibility:** Works with **Magisk**, **KernelSU**, and **APatch**.

## 🛠 Installation
1. Download the latest `zip` from the [Releases](#) section.
2. Open your root manager (Magisk / KernelSU / APatch).
3. Select **Install from storage** and pick the zip file.
4. Reboot your device.
